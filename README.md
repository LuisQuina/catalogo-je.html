# Catálogo Digital J&E - Botas y Botines

Plataforma ágil de consulta de inventario para J&E, diseñada para facilitar la búsqueda de productos y la gestión de leads comerciales a través de WhatsApp.

## Características Técnicas
- **Frontend**: HTML5, CSS3 y JavaScript vanilla.
- **Búsqueda**: Motor de búsqueda alfanumérico en tiempo real con filtrado dinámico.
- **Identidad de Marca**: Interfaz personalizada con branding integrado.
- **Flujo de Conversión**: Integración directa con WhatsApp (`wa.me`) para consultas rápidas.
- **Optimización**: Diseño responsive con adaptación automática para dispositivos móviles.

## Estructura del Proyecto
- `index.html`: Estructura principal y lógica de renderizado del catálogo.
- `catalogo_botines.json`: Base de datos de productos (fuente de verdad).
- `assets_botines/`: Recursos estáticos (imágenes de productos y logo de la marca).

## Despliegue
El proyecto está optimizado para entornos de servicios en la nube (PaaS) como **Railway**, aprovechando la entrega de sitios estáticos de alto rendimiento.

## Instrucciones de Mantenimiento
- Para actualizar el catálogo: Modificar el archivo `catalogo_botines.json`.
- Para cambiar la identidad visual: Actualizar el archivo `logo_je.png` en la carpeta `assets_botines`.
